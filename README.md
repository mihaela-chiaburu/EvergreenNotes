# EvergreenNotes

## Overview

EvergreenNotes is a Personal Knowledge Management (PKM) platform inspired by the concepts of Digital Gardens, Zettelkasten, Second Brain, and Active Recall learning. The platform aims to help users transform passive information consumption into an active knowledge-building process by capturing, connecting, revisiting, and expanding information over time.

---

## The Problem

Modern users consume massive amounts of information every day through articles, videos, social media, and online courses.
This often leads to:

* Information overload
* Doomscrolling
* Poor information retention
* Fragmented attention
* Passive content consumption

While tools such as Notion, Obsidian, and Anki solve parts of the problem, users often need multiple applications to manage information effectively.

---

## Features

### Knowledge Capture

* Create and edit notes
* Rich text content support
* Search and filtering
* Soft delete functionality

### Knowledge Organization

* Link notes together
* Build interconnected knowledge structures

### Knowledge Graph

* Interactive graph visualization using Cytoscape.js and D3.js

### Browser Capture Extension

* Capture content from articles and videos

### AI-Powered Garden Care

* AI-generated review questions using DeepSeek
* Encourage active recall

### Spaced Repetition

* Review scheduling system
* Notes reappear after predefined intervals

### Social Features

* Explore public gardens
* Follow other users
* Discover new ideas and knowledge networks
